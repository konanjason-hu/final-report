### 一、 預計完成功能
1. 自訂時間輸入：程式開始時，由使用者輸入預計倒數的秒數。
2. 大字體/置中顯示：利用 locate(x, y) 將倒數數字顯示在螢幕中央。
3. 色彩警示系統：
   - 剩餘時間 > 10秒：顯示綠色 (GREEN)。
   - 剩餘時間 6~10秒：顯示黃色 (YELLOW)。
   - 剩餘時間 < 5秒：顯示紅色 (RED) 並開始閃爍。
4. 自動優化介面：倒數期間隱藏游標 (hidecursor)，避免閃爍干擾。
5. 結束提示：時間歸零時，清空螢幕並變更視窗標題為「Time's Up!」。


### 二、 核心技術規劃
   - 畫面清理：使用 cls() 在每一秒更新數字前清空舊畫面。
   - 精確定位：使用 locate(x, y) 確保數字不會一直往下捲動，而是固定在同一位置。
   - 延遲控制：使用 msleep(1000) 來精確控制每一秒的間隔。
   - 顏色切換：使用 setColor(color) 根據剩餘秒數判斷式來改變文字顏色。


### 三、 預期執行流程
1. 呼叫 saveDefaultColor() 儲存原始設定。
2. 提示輸入秒數後，呼叫 hidecursor() 進入倒數模式。
3. 進入 while 迴圈進行倒數：
   - 計算剩餘時間。
   - cls() 清除螢幕。
   - 根據剩餘秒數 setColor()。
   - locate() 到螢幕中央並輸出數字。
   - msleep(1000) 暫停一秒。
4. 結束後 resetColor() 並 showcursor() 恢復正常狀態。
